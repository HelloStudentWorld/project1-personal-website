User Stories

Checking Out My Skills

As a recruiter, I want to see what skills the applicant has, so that I can decide if they're a good fit for the job.

Acceptance Criteria:
  • Easy-to-Find Skills Section.
  • Organized Skill Categories.
  • Detailed Skill List.
  • Looks Good on All Devices.
  • Visual Touches.
Showing Off My Projects

As a prospective employer, I want to see the projects the applicant has worked on, so that I can understand what they can do.

Acceptance Criteria:
  • Accessible Projects Section.
  • Cool Project Displays.
  • More Info on Projects.
  • Responsive Layout.
Getting in Touch with Me

As a potential employer, I want to contact them easily through a form, so that we can talk about job opportunities.

Acceptance Criteria:
  • Easy-to-Find Contact Section.
  • Simple Contact Form.
  • Form Works Correctly.
  • Confirmation Message.
