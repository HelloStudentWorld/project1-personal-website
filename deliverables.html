Overview

This project is a responsive web app designed to showcase my professional journey, projects, and expertise in the field of automotive software engineering. This web app serves as a central hub for user engagement, providing prospective employers/recruiters with a comprehensive overview of my background, skill set, and portfolio of projects, along with an easy/intuitive contact form for direct communication.

How to Use

Users can scroll up and down the single-page layout to quickly access the necessary information or utilize the sticky navbar for fast, section-specific navigation. This layout is designed to ensure a user experience that is clear and organized.

Key Sections in this Web App Include:

Homepage:

Hero section with my headshot, name, and tagline.

Sticky Navigation Bar linking to About, Skills, Projects, Contact, and Deliverables sections, allowing quick navigation.

About Section:

Mission statement outlining my goals.

Skills Section:

Categorized skills in Front-End, Back-End, and Tools with visual indicators of proficiency.

Grid layout showcasing project images, titles, descriptions, and external links.

Technologies Used

HTML5:

Used to structure the web app, creating the layout for sections like the navbar, skills, and contact form, ensuring the app is well-organized.

CSS3:

Applied for styling and visual design, enabling the web app to have a modern look and responsiveness across different devices.

Bootstrap 5:

Utilized for responsive design and pre-built components like grids and buttons, simplifying layout creation and ensuring mobile-friendliness.

JavaScript:

Enabled interactive features such as dynamic project listing and smooth scrolling for a better user experience.

Formspree:

Integrated for form submission handling, allowing the contact form to work without building a custom backend.

Google Fonts:

Used to apply the Montserrat font, ensuring a clean and modern typography across the web app.

Responsive Design and Media Queries:

Implemented for responsive design, ensuring the web app adjusts smoothly across different screen sizes.

Version Control with Git and GitHub:

Employed for version control and project hosting, providing code management and easy sharing with prospective employers.

User Stories

Checking Out My Skills

As a recruiter, I want to see what skills the applicant has, so that I can decide if they're a good fit for the job.

Acceptance Criteria:
  • Easy-to-Find Skills Section.
  • Organized Skill Categories.
  • Detailed Skill List.
  • Looks Good on All Devices.
  • Visual Touches.
Showing Off My Projects

As a prospective employer, I want to see the projects the applicant has worked on, so that I can understand what they can do.

Acceptance Criteria:
  • Accessible Projects Section.
  • Cool Project Displays.
  • More Info on Projects.
  • Responsive Layout.
Getting in Touch with Me

As a potential employer, I want to contact them easily through a form, so that we can talk about job opportunities.

Acceptance Criteria:
  • Easy-to-Find Contact Section.
  • Simple Contact Form.
  • Form Works Correctly.
  • Confirmation Message.
